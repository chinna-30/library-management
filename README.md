## 🎯 Objective

The objective of this project is to automate and simplify library operations by managing books, users, and book transactions efficiently. It helps reduce manual work, improve record accuracy, and maintain data permanently using file handling in C.

## 📌 Key Features

- 📖 Book Management (Add, View, Search, Modify, Delete)
- 👤 User Management
- 🔄 Book Issue & Return System
- 💾 File-Based Data Storage
- 🖥️ Menu-Driven Interface
- 📚 Linked List Implementation
